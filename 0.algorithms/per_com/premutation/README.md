# 순열

```
nPr
```

순열이란 서로 다른 n개중에 r개를 선택하는 경우의 수를 의미합니다. <br/>
순서를 고려합니다 <br/>

## 공식

```
nPr = n! / (n - r)!
```

## 증명

nPr 이라고 할때 , n개만큼 중에 r개만큼을 뽑은, 순서를 고려한 경우의 수를 구해야 한다. <br/>
첫번째로 뽑을 수 있는 수는 n 개이며, 두번째로 뽑을 수 있는 수는 n-1개이다. <br/>
r번째로 뽑을 수는 n+1-r 개이므로, 식으로 표현하면 아래와 같다.

```
nPr = n * (n - 1) * (n - 2) ... * (n + 1 - r)
    = !n / (n - r)!
```
